# Local Development

Requirements: `npm`

## Setting up the backend

Clone the repository and go to intall location via terminal
```
cd server
npm install nodemon
npm run dev
```
## Setting up the frontend

```
cd client
npm install next
npm run dev
```
Visit `https://localhost:3000/`
